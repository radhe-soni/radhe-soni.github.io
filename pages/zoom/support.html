Frequently Asked Questions

How do I install the extension?

You can install our extension directly from the Zoom Marketplace. Navigate to the Marketplace, search for our extension, and click "Install".

Why can't I see the extension in my meeting?

Make sure you have the latest version of Zoom installed and that you've granted all necessary permissions for the extension.

How do I update the extension?

Updates are typically automatic through the Zoom Marketplace. If you need to manually update, visit the Marketplace and click "Update".

Troubleshooting

  • Ensure you're using a supported browser (Chrome, Firefox, Edge)
  • Check that you've granted all required permissions
  • Try refreshing the Zoom client or restarting your computer
  • Verify you have a stable internet connection
  • Make sure you're using the latest version of the Zoom client
